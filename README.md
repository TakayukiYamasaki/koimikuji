恋みくじ Webアプリ（教材用）

このリポジトリは、HTML / CSS / JavaScript の基礎を学ぶための教材です。
高校生（プログラミング未経験）を対象に、段階的に完成度を高めていく構成になっています。

⸻

📁 プロジェクト構成

koimikuji/
├─ step0/  … HTMLのみ（表示の基本）
├─ step1/  … JavaScriptで動きを追加
├─ step2/  … 完成形（画面遷移＋全画面アニメーション）
└─ README.md


⸻

🟢 step0：HTML入門

学習ポイント
	•	HTMLの基本構造
	•	タグの役割（h1 / button / p）

できること
	•	画面にタイトルとボタンが表示される
	•	まだ動きはない

⸻

🟡 step1：JavaScript入門

学習ポイント
	•	ボタン操作（click）
	•	ランダム処理
	•	文字の書き換え

できること
	•	ボタンを押すとおみくじ結果が表示される

⸻

🔵 step2：完成形（おすすめ）

学習ポイント
	•	画面全体を使った演出
	•	setTimeout による待ち時間処理
	•	ページ遷移

できること
	•	ボタンを押す
	•	全画面アニメーションが表示される
	•	結果ページに移動
	•	最初の画面に戻れる
